### Install

### Config file
* Make sure there exists a config file.

$ `cd ~/.config`

$ `mkdir parser`

$ `touch parser.json`

* This is how `parser.json` should look like. Add your template destination like shown below.
```json
{
   "Template" : "/home/convict/Dropbox/myfiles/cplib/temp.cpp",
   "lang" : ""
}
```
